Reading

1. Read one data attribute

HTML:
JS: 
        var task1 = $('#task1').data('hello');
        console.log(task1);

2. Read multiple attributes

HTML:
JS: 
        var task2 = $('#task2').data();
        console.log(task2.one + ' ' + task2.two);

3. Read from multiple sources

HTML:
JS: 
        var task3 = $('.task3').data('hello');
        for(var i = 0; i < task3.length; i++){
            console.log('Hello ' + task3[i]);
        }
